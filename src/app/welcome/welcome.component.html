Welcome to Online Test Apllication

This application enables users to take online tests, review them, and display the results at the same time.

Quiz Rules:

  1. Each correct answer gives you 10 points and each incorrect answer will result in -10 points.
  2. You will have 60 seconds to answer each question.
  3. Refreshing the Page will reset the Quiz and you have to start from the beginning.
